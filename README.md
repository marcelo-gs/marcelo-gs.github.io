## 📁 Como baixar o projeto

```bash

# Clonar o repositorio
$ git clone https://wwww.github.com/JavaSoares/javasoares.github.io

# Entrar no diretorio 
$ cd JavaSoares

# Instalar as dependências
$ yarn install

# Iniciar o projeto
$ yarn start
```
